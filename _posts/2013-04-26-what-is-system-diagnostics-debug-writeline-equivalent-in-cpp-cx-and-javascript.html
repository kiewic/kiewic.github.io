In Windows Store Apps, to print something to the Output &gt; Debug window use:

In C#/VB:

System.Diagnostics.Debug.WriteLine("Hello monkey!");

In JavaScript:

console.log("Hello monkey!");

In C++/CX:

OutputDebugString(L"Hello monkey!\r\n");
